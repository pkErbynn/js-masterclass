## Bankist
A minimalistic javascript-based banking application with features like;
- transfere funds (attaches date)
- request loan (attaches requested date)
- display statements
- sort statement
- close account

### User pin
Experiment with this credentials:
```js
user1: js        user2: jd
pin1: 1111       pin2: 2222
```
